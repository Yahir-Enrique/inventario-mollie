# Inventario Mollie Aspen

Prototipo con backend Node.js + Express + SQLite.

## Instalar y ejecutar

1. Abrir terminal en la carpeta del proyecto.
2. Ejecutar:
   ```bash
   npm install
   npm start
   ```
3. Abrir en el navegador:
   ```
   http://localhost:3000
   ```

## Roles
- El primer usuario registrado se convierte en `admin`.
- Los siguientes usuarios se crean como `cajero`.

## Nota
La aplicación usa SQLite para que puedas entregar un ZIP con DB local y backend.
